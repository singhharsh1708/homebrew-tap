# homebrew-tap

Homebrew tap for [Kitbash](https://github.com/singhharsh1708/kitbash) — the package manager and compiler for AI agent skills.

```bash
brew install singhharsh1708/tap/kitbash
```

or

```bash
brew tap singhharsh1708/tap
brew install kitbash
```

Then `kitbash --version`. Same CLI as `npm install -g kitbash`.
